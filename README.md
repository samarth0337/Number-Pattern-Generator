📊 Number Pattern Generator
📌 Description

The Number Pattern Generator is a Python-based mini project that generates different number patterns using loops and functions. It is a menu-driven program where users can select various patterns and specify the number of rows to display.

🎯 Features
Multiple pattern generation:
Number Pyramid
Inverted Pyramid
Floyd’s Triangle
Pascal’s Triangle
Menu-driven interface
User input for dynamic pattern size
Modular code structure (packages & functions)
Input validation
🧠 Concepts Used
Nested loops
Functions
Conditional statements
Python modules & packages
Basic mathematical logic
